Integrate Notion databases as customizable forms in WordPress. Connect your Notion workspace with your WordPress site seamlessly.

== Description ==

Notion Forms allows you to transform your Notion databases into fully functional WordPress forms. This plugin creates a bridge between your WordPress site and Notion workspace, enabling you to collect form submissions directly into your Notion databases.

= Key Features =

* Connect multiple Notion databases to WordPress
* Customize form appearance with built-in styling options
* Drag-and-drop field arrangement
* Custom success messages and redirects
* Responsive form design
* Support for multiple field types
* Real-time database sync

= How It Works =

1. Connect your Notion integration
2. Select the database you want to use
3. Customize your form appearance
4. Add the form to any page using shortcode
5. Collect submissions directly in your Notion database

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/notion-forms` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Notion Forms settings page to configure your Notion integration
4. Create your first form by connecting to a Notion database

== Frequently Asked Questions ==

= Do I need a Notion account? =

Yes, you need a Notion account and the ability to create an integration in your Notion workspace.

= How do I get my Notion API key? =

1. Go to https://www.notion.so/my-integrations
2. Create a new integration
3. Copy the API key provided
4. Paste it in the plugin settings

= Can I customize the form appearance? =

Yes, the plugin includes a styles page where you can customize colors, fonts, spacing, and other visual elements.

= Is it possible to have multiple forms? =

Yes, you can create multiple forms connected to different Notion databases.

== Privacy Policy ==

This plugin connects to Notion's API to sync form data. All data is processed according to Notion's privacy policy and your WordPress site's privacy policy. No additional personal data is collected by the plugin itself beyond what users submit through the forms.
